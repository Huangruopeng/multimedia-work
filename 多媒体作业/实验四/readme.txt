其中webcode文件夹中是整个网页所需要的文件，包括素材，代码，数据库，环境文件等。
要想运行整个网页代码：
1、需要安装WampServer软件，下载地址文档中有。
2、然后将webcode文件夹中所有文件（注意不是文件夹）复制到该软件安装目录下的www文件夹下。如果遇到重名全部替代。
3、然后在浏览器中输入localhost/phpmyadmin 进入数据操作界面，新建一个数据库。
4、将数据库multimedia_db 导入之前新建的数据库。
5、更改connect.php中信息（就是密码用户名以及新建数据库的名字），使得后台与数据库连接。
6、在浏览器中输入localhost、即可访问网页。

素材文件夹里只包含了部分网页的图片。
